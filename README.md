# Multiplayer Replication Framework (WIP)

A Godot plugin that provides a framework for world state synchronization across multiple peers in a multiplayer game via
the network.

This plugin is built on top of Godot's built-in high-level multiplayer API, and is intended to be a more feature-rich
and efficient a replacement for Godot's built-in multiplayer nodes (`MultiplayerSynchronizer` and `MultiplayerSpawner`),
while still being compatible with other scripts that rely on the high-level multiplayer API.

> This project is still a work in progress in its early stages of development and is not ready for use yet.
